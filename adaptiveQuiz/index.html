Welcome to SecAware AI

This tool provides AI-powered security training using Google Gemini. It includes:

  • Risk Profile — Estimate user awareness level
  • Adaptive Quiz — Difficulty changes based on answers
  • Phishing Lab — Generate & analyze phishing scenarios
  • Policy → Training — Convert security policies into micro-training
